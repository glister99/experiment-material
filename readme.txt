How to use: Open the index.html with the browser of your choice. 
Compatibility: The prototype should work for most browsers, but only tested using Google Chrome.

The csv files:
- consent.csv registers the responses to the consent form;
- master.csv registers the dependent variables of the study;
- left.csv handles group assignment, it keeps track of the numbers of slots left.

If you get stuck on the "Requirements" page, and the console gives a CORS error, please try this: https://www.youtube.com/watch?v=nx8E5BF0XuE&ab_channel=dcode
